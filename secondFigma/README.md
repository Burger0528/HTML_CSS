# Julia Huang Portfolio

This project is a responsive portfolio layout built with HTML and CSS.
It focuses on modern layout techniques using Flexbox and CSS Grid.

## Features
- Two-column desktop layout
- Responsive behavior for tablet and mobile
- Flexbox for alignment
- CSS Grid for structured sections
- Clean and minimal design

## Technologies
- HTML5
- CSS3 (Flexbox & Grid)

## Responsive Design
- Desktop: two columns
- Tablet: stacked layout
- Mobile: single column with resized text

## Usage
Open `index.html` in your browser to view the project.
